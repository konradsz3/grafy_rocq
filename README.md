# Podstawowe zagadnienia teorii grafów w Rocq

W ramach projektu omówiliśmy podstawowe zagadnienia z obszaru kombinatoryki i teorii grafów w systemie Rocq, ze szczególnym uwzględnieniem grafów nieskierowanych, bez pętli i wielokrotnych krawędzi oraz grafów dwudzielnych.

## Zrealizowane zagadnienia
Na kod składają się następujące elementy:
- [x] Definicja typu **Graf** pozwalającego modelować grafy nieskierowane, bez pętli i wielokrotnych krawędzi o dowolnym zbiorze wierzchołków z `nat`.
- [x] Funkcja obliczająca stopień danego wierzchołka w grafie.
- [x] Definicja dwudzielności grafu.
- [x] **Przykłady:** dowody dwudzielności dla grafów $K_{3, 3}$ i $K_{1, 6}$.
- [x] Definicja dobrego klasycznego kolorowania wierzchołków grafu.
- [x] Dowód twierdzenia o dwukolorowalności grafów dwudzielnych.
